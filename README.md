# Gulp Example/Starter App

Optimize your HTML, CSS, JS and Image files with gulp

### Version
1.0.0

### Folder
Javascript files - src/js folder
CSS files - src/js folder
HTML files - src/
Image files - src/images/


##DevDependencies
gup
gulp-imagemin
gulp-concat
gulp-uglifycss
gulp-minify
gulp-htmlmin
gulp-notify
gulp-clean
gulp-runsequence

## Install Dependencies
```bash
npm install
```

## Run all tasks
```bash
gulp
```

## Watch all tasks/files
```bash
gulp watch
```
